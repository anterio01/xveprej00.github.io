Databázové systémy

Předmět

Předmět Databázové systémy jsou o základech databázových systémů. Opakuje se zde vytváření konceptuálního modelování a teorie relačního modelu dat, následně i návrh relační databáze z konceptuálního modelu, který je součást projektu. Dále se studenti naučí jazyk SQL, transakce, architektury DBS. Vyzkouší si činnosti administrátora databáze, pokud o optimalizaci a zotavení po poruše.

Projekt

Databázové systémy nabízí projekt na vytvoření databáze. Projekt je rozdělen na 4 fáze, kdy v první se jedná jen o vytvoření konceptuálního modelu a diagram užití. V dalších částech je projekt o vytvoření skriptu pro databázi s cizímy klíči podle jejich návrhu, jednoduché dotazy, složitější dotazy, procedury, optimalizace.

První část

Vytvoření konceptuálního modelu databáze je základ pro vytváření databází, jelikož je to prvotní a snadno upravitelný návrh, ze kterého se celá databáze staví.

Třetí část

Předposlední část dále rozvíjela stejný SQL skript. Cílem bylo vytvořit selects nad tabulkami. Mimo jednoduché a přímé selects přesně podle například specifického ID, tak bylo požadováno využít i ORDER BY, GROUP BY, EXISTS.

... /* A simple select that picks out authors and their book titles */ SELECT jmeno, prijmeni, titul FROM AUTOR JOIN KNIHA ON AUTOR.ID_autora = KNIHA.ID_autora JOIN PUBLIKACE ON KNIHA.ID_publikace = PUBLIKACE.ID_publikace; /* A simple select that picks out books that are/were loaned and their state */ SELECT titul, stav_pujcky FROM PUJCKA JOIN PUBLIKACE ON PUJCKA.ID_publikace = PUBLIKACE.ID_publikace; /* A simple select that picks out books that are reserved and their date of reservation */ SELECT titul, datum_rezervace FROM REZERVACE JOIN PUBLIKACE ON REZERVACE.ID_publikace = PUBLIKACE.ID_publikace; /* A simple select that picks out readers that have/had a loan */ SELECT jmeno, prijmeni, titul FROM CTENAR JOIN OSOBA ON CTENAR.ID_osoby = OSOBA.ID_osoby JOIN PUJCKA ON PUJCKA.ID_ctenare = CTENAR.ID_osoby JOIN PUBLIKACE ON PUBLIKACE.ID_publikace = PUJCKA.ID_publikace ORDER BY prijmeni ASC; /* A select that picks out readers and the amount of their loans */ SELECT jmeno, prijmeni, COUNT(*) AS pocet_pujcek FROM PUJCKA JOIN OSOBA ON PUJCKA.ID_ctenare = OSOBA.ID_osoby GROUP BY jmeno, prijmeni; /* A select that picks out the bindings of books and counts how many of them have either softcover or hardcover */ SELECT vazba, COUNT(*) AS pocet_knih FROM KNIHA WHERE vazba IN ('měkká', 'tvrdá') GROUP BY vazba; /* A select that picks out names of readers with atleast 1 loan */ SELECT jmeno, prijmeni FROM OSOBA WHERE EXISTS ( SELECT 1 FROM PUJCKA WHERE PUJCKA.ID_ctenare = OSOBA.ID_osoby ); /* A select that picks out book titles that have been loaned atleast once */ SELECT titul FROM PUBLIKACE WHERE ID_publikace IN ( SELECT ID_publikace FROM PUJCKA ); ...

SET SERVEROUTPUT ON; /* A procedure, that, when invoked, prints out the active loans of a specific reader */ CREATE OR REPLACE PROCEDURE vypis_aktivni_pujcky_ctenare ( p_ctenar_id IN CTENAR.ID_osoby%TYPE ) IS -- Cursor to select active loans for the given reader CURSOR c_aktivni_pujcky IS SELECT p.ID_publikace, p.titul, pj.datum_pujcky, pj.stav_pujcky FROM PUJCKA pj JOIN PUBLIKACE p ON pj.ID_publikace = p.ID_publikace WHERE pj.ID_ctenare = p_ctenar_id AND pj.stav_pujcky IN ('vypujceno', 'zpozdeno'); -- Variables with the usage of %TYPE v_titul PUBLIKACE.titul%TYPE; v_datum_pujcky PUJCKA.datum_pujcky%TYPE; v_stav PUJCKA.stav_pujcky%TYPE; v_id_pub PUJCKA.ID_publikace%TYPE; BEGIN -- Looping through the extracted rows FOR pujcka_rec IN c_aktivni_pujcky LOOP -- Tying the variables together v_id_pub := pujcka_rec.ID_publikace; v_titul := pujcka_rec.titul; v_datum_pujcky := pujcka_rec.datum_pujcky; v_stav := pujcka_rec.stav_pujcky; DBMS_OUTPUT.PUT_LINE('ID publikace: ' || v_id_pub || ', Titul: ' || v_titul || ', Půjčeno dne: ' || TO_CHAR(v_datum_pujcky, 'YYYY-MM-DD') || ', Stav: ' || v_stav); END LOOP; END; / ... /* EXPLAIN PLAN before index creation */ EXPLAIN PLAN FOR SELECT z.typ_zanru, COUNT(k.ID_publikace) AS pocet_knih FROM ZANR z JOIN KNIHA_ZANR kz ON z.ID_zanru = kz.ID_zanru JOIN KNIHA k ON kz.ID_publikace = k.ID_publikace GROUP BY z.typ_zanru ORDER BY pocet_knih DESC; /* Show plan */ SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY); /* Create index to optimize */ CREATE INDEX idx_knihazanr_zanr ON KNIHA_ZANR(ID_zanru); CREATE INDEX idx_knihazanr_publikace ON KNIHA_ZANR(ID_publikace); /* EXPLAIN PLAN after index creation */ EXPLAIN PLAN FOR SELECT z.typ_zanru, COUNT(k.ID_publikace) AS pocet_knih FROM ZANR z JOIN KNIHA_ZANR kz ON z.ID_zanru = kz.ID_zanru JOIN KNIHA k ON kz.ID_publikace = k.ID_publikace GROUP BY z.typ_zanru ORDER BY pocet_knih DESC; /* Show plan */ SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY); ...
